# SPX Toolkit (Local)

Site estático para rodar no GitHub Pages e salvar config em localStorage.

## Deploy (GitHub Pages)
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main / root
4. Salvar

URL fica tipo:
https://SEUUSUARIO.github.io/spx-toolkit-local/

## Config
- Station name
- Headers JSON (sem cookie)

## Observação
Se o SPX bloquear CORS, o browser não permite ler respostas sem extensão.
